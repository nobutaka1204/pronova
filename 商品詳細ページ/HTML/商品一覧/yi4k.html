YI TechnologyとXiaomiについて

YI Technologyは中国上海を本社拠点として2014年に創業し革新的なカメラ製品を世界中で開発販売する企業です。創業2年でユーザー数が450万人を超えるほどまでに成長したグローバル企業です。YI 4K Action CameraはもともとXiaomi YI2としてXiaomiから販売されていたものとほとんどスペックや外観までほとんど同じです。またXiaomiは現在多くの中国メーカーに投資を加速させています。そんなYI Technologyはあの中国のアップルと言われるXiaomiから出資を受けているようなのでXiaomiグループの企業と言えそうです。おそらく多くの技術や人員、資本をXiaomiから受け継いでいると思います。 ちなみにYIは「イー」と発音しYoung Innovatorの略です。 創業してから数年で世界の先端カメラの代表格になるなど今後が期待できる企業です。

YI2【YI 4K Action Camera】の基本情報

 

YI2(YI 4K Action Camera)は最新のイメージセンサーやイメージプロセッサをはじめあらゆる部分に最先端の技術を採用しています。最先端の技術を用いることで最大の使いやすさと凡用性を提供しています。また、ディスプレイを搭載することで簡単に即座に撮影可能でなおかつその場で撮影内容を確認することができるようになり使い易さが向上しています。 YI Technologyのオリジナル アプリと、高速5 GHz Wi-Fiにより使用して、編集、フィルタ適用、音楽の追加、ソーシャルメディアへの即時共有できます。 そんなYI 4K Action Cameraの基本的な情報を紹介していきます。

YI2はついに4K撮影に対応したアクションカメラでの別次元の画質に

 

今回YI 4K Action Cameraでは4K 30fps で1080pの4倍の解像度を実現しました。その他にも1080pで120fps、720pで240fpsなどの高解像度の映像を撮影することが可能です。また12MPの撮影時は、 タイムラプス（Timelapse） スローモーション（Slow Motion） タイマー（Timer） バースト（Burst） などの撮影モードを選択できます それぞれのモードで撮影した映像のレビューが下にありますので見てみてください。それぞれ使い勝手の良いモードになっていることがわかると思います。

YI2はゴリラガラス液晶網膜タッチスクリーン搭載で直感的な操作が可能！

 

YI 4K Action Cameraに新たに搭載されたディスプレイは330ppiというカメラ用のディスプレイとは考えられないほどの高解像度、さらに高感度の網膜タッチスクリーンです。 さらに高解像度なだけでなく160度の角度から見ることができるので、仲間たちとYI 4K Action Cameraを囲んで見るのにも適しています。スクリーンの大きさも640×360と十分な大きさになっているのでその場でのプレビューや撮影が楽しくなりそうです。 また、YI 4K Action CameraではXperiaなどでも採用されているゴリラガラスを採用しているので耐久性の面でも安心できる仕上がりになっています。

電子式動体ブレ補正機能（EIS）を搭載したYI2

 

そしてついにYI 4K Action Cameraでは手振れ補正機能が加わりました。これによりアクションカメラにはつきものの映像がぶれるという心配が大幅に軽減されています。 手振れ補正の仕組みは電子式動体ブレ補正機能や電子画像安定化機構などと訳されるEISという技術を使っています。この技術では3軸ジャイロスコープと3軸加速度センサーを利用してYI2本体の位置と動きの変化を検出しYI2の動きや揺れ衝撃から受けたブレをクロスボーダー補正アルゴリズムによりブレを補正することができます。

360度無指向性デュアルマイク搭載で周りの音を逃さない！

 

YI 4K Action Cameraは360度無指向性デュアルマイクを利用し48kHzのステレオサウンドで録音することができます。とても高感度でSN比の高いマイクを搭載しているので、環境音のうるさい場所でもクリアな音を録音できます。また、最適化されたAACエンコーダーが備わっていて細かな音まで録音可能です。

YI2は1回の充電で4K/30fpsビデオを2時間以上撮影可能！

 

YI 4K Action Cameraは充電式の1400mAh高電圧4.4Vリチウムバッテリーを使用しています。大容量のバッテリーなので安心して長時間の撮影にも耐えられます。最大8時間の待機時間に達して、1回の充電で最大120分間の4K 30fpsでのビデオが可能です。

YIカメラアプリならその場で確認し編集さらには共有もできる！

 

YI Technologyが開発したYIカメラアプリではその場で素早くビデオや画像を編集したりフィルターにかけたり音楽を付けたりすることができます。また映像をfacebookやyoutubeなどのsnsや動画投稿サイトにその場でアップロードすることが可能です。 YIカメラアプリの特徴は直感的な操作のみで高品質のビデオ編集を行うことができることです。趣味のための映像編集でしたらパソコンがなくても十分足りると思います。

高速WIFI接続技術に対応したYI2なら高画質動画も通信が楽

 

YI 4K Action Cameraでは通信用に半導体ソリューションBCM43340を採用しているので2.4GHzと5GHzの両方に対応しているのでバンドの干渉を防止することができます。また起動までに3秒しかかからないず30Mbpsのスピードで通信可能なので、4Kなどの高解像度の撮影にもしっかりと対応可能になっています。
